I. About This Guide

This guide is continuously being updated with the most recent information. Because of the nature of this project, the characteristics and/or functions of the robot and this guide are subject to change at any time without prior notice. If you have downloaded or printed a copy of this guide please refer to the version number on the cover page to ensure that you have the latest edition.

This guide is meant to provide guidance in the construction of a humanoid robot using parts that are readily available. You may choose to build your robot differently than what the recommendations in this guide provide. Many parts of this guide are designed to be flexible in order to allow parts to be built using whatever is at hand. Note that each chapter has been designed so that the construction process can be expedited by building parts of the robot in parallel. This makes teamwork a lucrative benefactor in the construction process.

If you have any ideas or suggestions for how this guide could be improved please send them to gunther@salvius.org.
