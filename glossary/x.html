Glossary - X

A glossary of terms commonly used in robotics, electronics, and programming.

Back to Glossary Index

X

Xenobot
A type of biological robot created from living cells, specifically frog cells. These programmable organisms represent a new form of life that can move, carry payloads, and self-heal, opening new possibilities for biomedical and environmental applications.
XML (eXtensible Markup Language)
A markup language used for encoding documents and data in a format that is both human-readable and machine-readable. In robotics, XML is often used for configuration files, data exchange between systems, and robot description formats like URDF.
XYZ Coordinates
A reference given to the most common names of the lines forming a Cartesian coordinate system. In robotics, XYZ coordinates define position in 3D space, where X typically represents left-right movement, Y represents forward-backward movement, and Z represents up-down movement.
