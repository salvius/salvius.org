Glossary - V

A glossary of terms commonly used in robotics, electronics, and programming.

Back to Glossary Index

V

Vision Guided
A control system where the trajectory of the robot is altered in response to input from a vision system.
Vision Sensor
A sensor which identifies the shape, location, orientation, or dimensions of an object using a visual feedback device, such as a camera.
Voltage Regulator
An electronic circuit that provides a stable output voltage regardless of changes in the input voltage or load conditions.
